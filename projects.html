Beschreibung:


Beim Erstellen der Coding-Time Github Page waren wir auf der Suche nach einem geeigneten "reset"-Stylesheet. Dabei kam uns die Idee selbst einen Stylesheet zu erstellen, welcher ueber die ueblichen CSS-resets hinausgeht und die gaengigen HTML Elemente auf ein vorsintflutliches Level zurueckbringt. Unser Ziel war es dass eine Website bei Nutzung fast wie eine Textdatei aussieht.


Fazit:


Auch wenn der praktische Anwendungsbereich fuer den wie-gott-sie-schuf.css gering ausfallen duerfte, eignete er sich doch als gute erste Fingeruebung. Man kann das ganze eher als kleines Experiment betrachten. We did it for the lolz.

bower install wie-gott-sie-schuf
